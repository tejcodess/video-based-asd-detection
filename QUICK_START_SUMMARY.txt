================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Python not found"
Solution: Install Python 3.11 and check "Add to PATH"

Issue: "TensorFlow not found"
Solution: Run setup_venv.ps1 again

Issue: "No GPU detected"
Solution: Training works on CPU (slower), or install NVIDIA drivers

Issue: "Out of memory"
Solution: Reduce batch size in recurrent_networks.py (line ~200)
          Change BATCH_SIZE = 625 to BATCH_SIZE = 256

Issue: "Dataset not found"
Solution: Check path in config.py matches your dataset location
